# Quick Start Guide

## Step 1: Save Your Images

You need to save 2 images:

### Logo Image
- The image with black background and gold "UF URBAN FASHION" text
- Save to: `public/images/logo/logo.png`

### Hero Image  
- The fashion model image in urban setting
- Save to: `public/images/hero/hero.jpg`

## Step 2: Start the Development Server

Open terminal in the `urban-fashion` folder and run:

```bash
npm run dev
```

## Step 3: Open in Browser

The terminal will show a URL like:
```
Local: http://localhost:5173
```

Click it or copy-paste into your browser.

## What You'll See

- **Home Page**: Full-screen hero with your logo and "ONLY FOR THOSE WHO DARE TO LEAD" text
- **Navbar**: Sticky navigation that shrinks on scroll
- **About Page**: Brand story with 4 value cards
- **Lookbook Page**: Gallery grid (will show placeholders until you add more images)
- **Blog Page**: 3 blog posts (will show placeholders until you add images)
- **Contact Page**: Contact form with social links

## Animations

- Logo scales from 1.8x to 1x as you scroll
- Smooth fade-in effects on all sections
- Hover effects on buttons and images
- Parallax scrolling on hero section
